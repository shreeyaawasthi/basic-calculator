# 🧮 Basic Calculator

A simple and responsive Basic Calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations and is suitable for beginners learning web development and JavaScript.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 📊 Percentage
- 🔢 Decimal Calculations
- ❌ Clear Display (C)
- ⌫ Delete Last Character (DEL)
- ⚠️ Error Handling for Invalid Expressions
- 📱 Responsive and User-Friendly Interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any web browser.
4. Start performing calculations.

No additional software or installation is required.

---

## 💻 How It Works

- HTML creates the calculator layout.
- CSS styles the calculator and buttons.
- JavaScript handles user input, performs calculations, and displays the result.

---

## 📸 Calculator Functions

| Button | Function |
|---------|----------|
| 0-9 | Enter numbers |
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Percentage |
| . | Decimal Point |
| C | Clear Display |
| DEL | Delete Last Character |
| = | Calculate Result |

---

## 📖 JavaScript Functions

### appendValue(value)
Adds the selected number or operator to the display.

### clearDisplay()
Clears all values from the calculator display.

### deleteLast()
Deletes the last entered character.

### calculate()
Evaluates the mathematical expression and displays the result. If the expression is invalid, it displays **"Error"**.
